W3C Invited Expert

CSS Color

Created implementations for css-color-4 and css-color-5.

  • worked together with the specification editors to improve the specification texts
  • contributed various new tests and fixes to WPT
  • created PostCSS transpilation plugins based on these implementations
  • still advocating for browser vendors to implement these features correctly

Media Queries

Advocated for media queries as a broad focus area for interop 2023.
I added various tests to WPT to illustrate the various interop issues and gaps in implementations.
This was accepted and all browsers now have greatly improved support for media queries.

Improved support for modern media query features in various developer tools.

  • media query parser
  • improved support for media range syntax in PostCSS
  • improved static analysis of media queries in Stylelint

CSS Tokenizer and Parsers

Created a specification compliant CSS tokenizer and various parsers.

  • CSS tokenizer
  • re-usable tests for CSS tokenizers
  • CSS parser algorithms
  • media query parser
These tools drive much of the new PostCSS plugins and are also used in projects like Stylelint.

Browser Compat

Regular contributor to MDN Browser Compat Data.
I also created web-tests as a way to quickly gather historical support data for various web features.

I advocated for something less biased and more inclusive than the first iteration of MDN/Google Baseline.
As part of this work I interacted with the community group to share my insights both as the maintainer of countless polyfills and as a web developer with a specific focus on supporting older browser versions.
The current definition (30 months since interop) aligns much more closely with end user needs and is much more inclusive.
